// Description of dataset: 
This dataset contains the data required to reproduce the results in the article "Rediscovering shallow-water equations from experimental data". The dataset consists of experimental recordings of shallow-water solitons generated in a simple wave flume, together with processed MATLAB data files derived from those recordings.

The release includes:
(1) raw TIFF image sequences for individual soliton recordings;
(2) processed MATLAB MAT files containing matrix-form data used in the analyses reported in the paper;

The raw TIFF image sequences are grouped into training and test cases. The processed MAT files are the exact processed versions used in the paper. These processed files were created from the raw image sequences using the preprocessing workflow described in the manuscript.

Related publication: Submitted for review

// File List: 
00_README.txt
    Plain-text documentation file describing the dataset, files, and methods.

Train01.zip to Train18.zip
    ZIP archives containing the raw TIFF image sequences for the 18 solitons used as the training set. Each ZIP file corresponds to one recorded soliton event.

Test01.zip to Test07.zip
    ZIP archives containing the raw TIFF image sequences for the 7 solitons used as the test set. Each ZIP file corresponds to one recorded soliton event.

Processed_mat.zip
    ZIP archive containing the processed MATLAB MAT files used in the paper. These files contain the processed matrix-form data derived from the raw videos and are the exact processed versions used in the reported analyses.

// Relationship between files, if important: 
The Train*.zip and Test*.zip files contain the raw image data. Processed_mat.zip contains processed data derived from those raw image sequences.
